Elastic Search
==============

**1. Install**
- Require **JDK** => install **JDK**
```ruby
sudo apt-get update
sudo apt-get install openjdk-7-jre
java -version
```
- Install **EL**
+ Dowload [hehe](https://www.elastic.co/downloads/elasticsearch) (.deb version)
```ruby
sudo dpkg -i elasticsearch-[version].deb
sudo /etc/init.d/elasticsearch start
```
